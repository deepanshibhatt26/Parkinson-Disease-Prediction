Methodology

How the Prediction Engine Works

📥
1. GEO Fetch
Download both datasets. Auto-detect PD vs Control labels from metadata.
⚙️
2. Normalize
Log₂ + quantile normalization on both. Z-score per-gene for cross-dataset comparison.
🧬
3. DEA + Signature
Welch's t-test + BH FDR on GSE1. Extract top N signature genes (direction + FC).
🤖
4. Score + ML
Apply signature to each GSE2 sample. Signature match score + RF/SVM/LR ensemble → confidence.
📊
5. Report
Per-sample confidence score, pathway disruption, visualizations & downloadable report.
